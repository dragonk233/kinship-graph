# Kinship Graph · 亲族图谱

这是一个验证核心体验的本地网页原型：记录家族人物，从任意人物视角重新计算普通话和闽南语称呼。

## 运行

双击 `启动亲族图谱.command`，或在终端执行：

```bash
npm install
npm run dev
```

浏览器打开：<http://127.0.0.1:5173>

线上版本：<https://dragonk233.github.io/kinship-graph/>

## 当前可体验

- 内置16人的四代示例家族；
- 点击人物查看关系、称呼和推导路径；
- 点击“以此人为我”或双击人物切换主视角；
- 按住画布空白处自由拖动，使用滚轮或右下角按钮缩放；
- 点击“适应”自动显示全部人物；
- 所有人物的普通话称呼随主视角刷新；
- 常用闽南语称呼示范；
- 按姓名或称呼搜索；
- 完整关系录入：支持父母、配偶、子女、兄弟姐妹、祖孙、叔伯姑姨舅、侄甥和姻亲长辈等常用称谓。
- 精确连接：可任选已有亲人，以父母、子女或配偶的基础事实连接新人物，并在提交前预览实际关系链。
- 在人物详情中查看和移除直接关系；移除关系不会删除人物资料。
- 编辑亲人的姓名、性别、出生年份、所属支系和备注。
- 家谱修改会自动精简并保存到当前浏览器的 IndexedDB；不保存历史副本，也不写入头像数据。
- 支持导出紧凑 JSON 备份，并在其他浏览器中导入恢复；导入前会校验人物与关系数据。
- 点击闽南语称呼旁的“播报”，离线播放教育部官方真人发音。

## 闽南语音频来源

称呼发音来自教育部《臺灣台語常用詞辭典》的词条 MP3，按“创用 CC 姓名标示—禁止改作 3.0 台湾”授权使用。项目保留原始音频、不剪辑、不变速，并在界面中标注来源。

常见亲属关系直接播放对应的官方称呼；没有独立词条的复杂关系按关系路径逐段播放官方真人发音。关系引擎支持的父、母、夫、妻、子、女及兄弟姊妹十种基础边均有音频，因此任意已连通关系都有可播放结果，不会用普通话替代。

- 来源：https://sutian.moe.edu.tw/
- 授权：https://sutian.moe.edu.tw/zh-hant/piantsip/pankhuan-singbing/

头像功能代码暂时保留但默认关闭。家谱数据仅保存在当前浏览器；清理网站数据或更换设备不会自动同步，请定期使用“备份”导出 JSON 文件。闽南语词库也需要按具体地区和家庭习惯核对。
