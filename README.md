# 亲族图谱 · 最小可运行模型

这是一个验证核心体验的本地网页原型：记录家族人物，从任意人物视角重新计算普通话和闽南语称呼。

## 运行

双击 `启动亲族图谱.command`，或在终端执行：

```bash
npm install
npm run dev
```

浏览器打开：<http://127.0.0.1:5173>

## 当前可体验

- 内置16人的四代示例家族；
- 点击人物查看关系、称呼和推导路径；
- 点击“以此人为我”或双击人物切换主视角；
- 按住画布空白处自由拖动，使用滚轮或右下角按钮缩放；
- 点击“适应”自动显示全部人物；
- 所有人物的普通话称呼随主视角刷新；
- 常用闽南语称呼示范；
- 按姓名或称呼搜索；
- 添加当前主视角的父母、配偶或子女。
- 编辑亲人的姓名、性别、出生年份、所属支系和备注。
- 点击闽南语称呼旁的“播报”，通过闽南语 TTS 朗读；服务不可用时仅回退到设备原生 `nan-TW` 声音。

## 配置闽南语 TTS

项目已接入兼容 REST 的闽南语 TTS 服务端代理，密钥不会发送到浏览器。复制 `.env.example` 为 `.env.local`，填入服务商提供的正式端点、API 密钥、音色和模型后，重新启动即可。代理兼容直接返回 WAV/音频的接口，也兼容返回 `{ "audio_url": "..." }` 的接口。

Taigi AI Labs 支持台湾台语汉字和台罗输入，但正式端点、鉴权方式及模型名需向服务方申请，不能使用官网中的示例地址。拿到资料后按对方文档调整 `.env.local`；若请求字段与默认格式不同，再同步调整 `vite.config.ts` 中的请求体。

这是交互模型，不是最终数据版本。新增数据暂存在浏览器状态，刷新页面后重置；闽南语词库也需要按具体地区和家庭习惯核对。
