# Kinship Graph · 亲族图谱

这是一个验证核心体验的本地网页原型：记录家族人物，从任意人物视角重新计算普通话和闽南语称呼。

## 运行

双击 `启动亲族图谱.command`，或在终端执行：

```bash
npm install
npm run dev
```

浏览器打开：<http://127.0.0.1:5173>

线上版本：<https://dragonk233.github.io/kinship-graph/>

## 安装到手机桌面

线上版本支持 PWA 安装和离线使用。完整打开一次后，应用外壳、关系功能与闽南语音频会保存在本机。

- iPhone：使用 Safari 打开，点击“分享”→“添加到主屏幕”；
- Android：点击应用内“安装到桌面”，或使用浏览器菜单中的“安装应用”；
- 应用更新时会显示重新加载提示，本地家谱不会随版本更新被覆盖。

家谱仍只保存在当前设备的应用空间中。建议定期从“备份”导出或分享 JSON 恢复文件；卸载应用或清理网站数据前务必先备份。

## 当前可体验

- 首次打开只创建“我”这一个节点，不预填示例家族数据；
- 点击人物查看关系、称呼和推导路径；
- 在自由画布中单击人物进行选择，双击直接打开人物编辑框；
- 切换主视角只从所选人物的操作菜单进入；
- 按住画布空白处自由拖动，使用滚轮或右下角按钮缩放；
- 点击“适应”自动显示全部人物；
- 所有人物的普通话称呼随主视角刷新；
- 常用闽南语称呼示范；
- 按姓名或称呼搜索；
- 完整关系录入：支持父母、配偶、子女、兄弟姐妹、祖孙、叔伯姑姨舅、侄甥和姻亲长辈等常用称谓。
- 精确连接：可任选已有亲人，以父母、子女或配偶的基础事实连接新人物，并在提交前预览实际关系链。
- 在人物详情中查看和移除直接关系；移除关系不会删除人物资料。
- 编辑亲人的姓名、性别、出生年份、所属支系和备注。
- 人物档案支持别名、籍贯、支系、在世状态、逝世日期，以及迁居、求学、职业、婚姻等生平时间线。
- 亲子关系可区分亲生、收养和继亲；婚姻关系可记录婚姻中、离异、丧偶或曾为配偶及起止日期。
- 家谱修改会自动精简并保存到当前浏览器的 IndexedDB；历史版本同样只保存在本机，不写入头像数据。
- 支持在同一浏览器建立多份独立家谱，并在“家谱书架”中切换、重命名和删除。
- 每次修改前自动保存本机历史快照，最多保留最近 40 个版本，可随时恢复且不会覆盖恢复前状态。
- 支持导出紧凑 JSON 备份，并在其他浏览器中导入恢复；也可导出含 Mermaid 图和人物资料的 Markdown 文件直接查看与分享。
- 支持 GEDCOM 5.5.1 常用字段的导入导出；导入后可检查并合并同名、同生日的重复人物。
- 家事历自动汇总近期生日、整岁、纪念日和带日期的生平事件，并同时显示农历生日。
- 大型家谱可以只显示当前人物上下三层关系，也可按辈分、支系和法律关系筛选。
- 提供紧凑/详细打印版，可通过浏览器保存 PDF、隐藏人物备注，或直接导出可继续排版的 SVG 图谱。
- 点击闽南语称呼旁的“播报”，离线播放教育部官方真人发音。
- 可为任意“查看者 → 亲人”保存家庭实际叫法，同时保留系统标准称呼。
- “两人关系”同时给出任意两人的双向称呼与推导路径。
- 支持撤销本次会话中的最近修改，并检查重复关系、亲子循环和明显年龄风险。

## 闽南语音频来源

称呼发音来自教育部《臺灣台語常用詞辭典》的词条 MP3，按“创用 CC 姓名标示—禁止改作 3.0 台湾”授权使用。项目保留原始音频、不剪辑、不变速，并在界面中标注来源。

常见亲属关系直接播放对应的官方称呼；没有独立词条的复杂关系按关系路径逐段播放官方真人发音。关系引擎支持的父、母、夫、妻、子、女及兄弟姊妹十种基础边均有音频，因此任意已连通关系都有可播放结果，不会用普通话替代。

- 来源：https://sutian.moe.edu.tw/
- 授权：https://sutian.moe.edu.tw/zh-hant/piantsip/pankhuan-singbing/

头像功能代码暂时保留但默认关闭。本项目仍是纯静态网站，不包含账号、服务器数据库、云同步或多人协作；多份家谱、历史版本和人物时间线都只保存在当前浏览器。清理网站数据或更换设备不会自动同步，请定期使用“备份”导出 JSON 或 GEDCOM 文件。闽南语词库也需要按具体地区和家庭习惯核对。
