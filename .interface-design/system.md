# 亲族图谱界面设计系统

## 方向与感受

这是一个供家庭成员整理人物、核对关系和理解亲属称谓的家谱工具。界面应当像一本正在使用的现代族谱册：安静、可信、有纸本温度，但仍保持工具软件所需的紧凑和明确。

领域语言来自族谱、世代、支系、称谓、血缘链、婚姻链与印章。避免通用 SaaS 仪表盘、悬浮指标卡和无意义装饰。

## 视觉签名

“关系路径”是产品的核心签名：任何推导结果或关系写入都尽量显示为人物与基础事实组成的可读路径。朱砂用于标记当前视角、当前选择、关系路径方向和主要操作。

签名应出现在：

- 当前主视角印记；
- 画布人物与连接线；
- 称谓推导路径；
- 新增关系前的写入预览；
- 关系维护入口和选中状态。

## 色彩与材质

颜色来自宣纸米白、木版墨色、旧照片棕、朱砂印和族谱线灰。颜色必须承载结构、状态或关系含义，不使用装饰性渐变或多套强调色。

现有核心令牌：

```css
--paper: #f5f1e8;
--paper-raised: #faf7f0;
--paper-inset: #ede8dc;
--ink: #28251f;
--ink-2: #5e584e;
--ink-3: #888074;
--line: rgba(59, 52, 42, .15);
--line-soft: rgba(59, 52, 42, .08);
--cinnabar: #a53a2b;
--cinnabar-dark: #812a20;
--wood: #765844;
--blood: #8d4a3c;
--marriage: #8e7a5c;
```

## 层级与深度

采用轻边界和细微表面色差，避免厚重阴影。页面骨架由低透明度边界分隔；输入区域使用略深的 `paper-inset` 表现“承接内容”；弹窗使用 `paper-raised`，只允许弹窗和画布节点使用克制的柔和阴影。

表面层级：

1. 页面与侧栏：`paper`；
2. 画布或输入底：`paper-inset` 及同色系细微变化；
3. 人物卡、操作卡：`paper-raised`；
4. 弹窗：`paper-raised` + 低透明度大范围阴影。

## 字体

人物、称谓和标题使用宋体系，强化族谱与纸本文献感：`Songti SC`, `Noto Serif SC`, `Microsoft YaHei`, serif。

搜索、辅助说明、状态、元数据使用系统无衬线字体。路径和技术性关系序列使用等宽字体。层级不仅依赖字号，也要使用字重、字距和颜色区分。

## 间距与圆角

基础间距单位为 4px。常用间距为 8、12、16、20、24px；紧凑元数据允许 4 或 6px。禁止无理由出现不属于该尺度的随机间距。

圆角尺度：

- 小控件和标签：4–5px；
- 输入、按钮和紧凑卡片：6–7px；
- 人物卡：8px；
- 弹窗：10px。

## 核心组件模式

### 人物画布

- 人物卡显示姓名、动态称谓和必要的人物特征；
- 血缘使用实线，婚姻使用虚线；
- 当前视角用朱砂边框与“我”印记；
- 单击查看，双击切换视角；
- 画布支持拖动、缩放和适应全部人物。

### 关系录入

- 使用“关系分组 → 具体称谓 → 必要的连接人物”的顺序；
- 常用称谓必须转换成父母、子女或配偶等稳定基础事实；
- 未覆盖的关系通过“精确连接”录入，不不断扩张超长称谓列表；
- 提交前显示实际写入的关系链；
- 缺少必要中间人物时，给出原因并禁用提交；
- 移动端关系选项使用两列，桌面端最多三列。

### 关系维护

- 人物资料和人物关系分开维护；
- 关系维护只显示实际保存的基础连接，不把推导称谓伪装成可编辑事实；
- 移除关系不删除人物，并在操作后明确反馈；
- 删除人物必须使用单独的危险确认流程。

### 状态与反馈

- 交互控件必须具备默认、悬停、聚焦、选中、禁用状态；
- 朱砂用于主要操作、选择、警告和破坏性操作，但通过背景透明度区分强度；
- 保存、空数据、缺少连接人物和录入失败都必须有明确状态；
- Toast 简短描述结果，不承担需要长期阅读的信息。

## 响应式规则

- 820px 以下切换为纵向页面：画布、人物列表、人物详情依次排列；
- 弹窗最大高度为视口减 32px，内部滚动，页面背景保持固定；
- 520px 以下表单和关系选项改为适合触控的两列或单列；
- 不为适配而隐藏关键关系说明、错误提示或提交状态。

## 设计禁区

- 不使用通用后台侧栏模板或 KPI 卡片；
- 不为每种支系或称谓分配装饰性色彩；
- 不直接保存“叔叔”“表姐”等可推导称谓作为事实关系；
- 不把人物编辑、关系编辑和删除人物混成一个无层级长表单；
- 不使用强边框、厚阴影、大圆角或弹跳动画；
- 不让关系结果只有结论而没有推导路径。
