# 亲族图谱 · 最小可运行模型

这是一个验证核心体验的本地网页原型：记录家族人物，从任意人物视角重新计算普通话和闽南语称呼。

## 运行

双击 `启动亲族图谱.command`，或在终端执行：

```bash
npm install
npm run dev
```

浏览器打开：<http://127.0.0.1:5173>

## 当前可体验

- 内置16人的四代示例家族；
- 点击人物查看关系、称呼和推导路径；
- 点击“以此人为我”或双击人物切换主视角；
- 按住画布空白处自由拖动，使用滚轮或右下角按钮缩放；
- 点击“适应”自动显示全部人物；
- 所有人物的普通话称呼随主视角刷新；
- 常用闽南语称呼示范；
- 按姓名或称呼搜索；
- 添加当前主视角的父母、配偶或子女。
- 编辑亲人的姓名、性别、出生年份、所属支系和备注。

这是交互模型，不是最终数据版本。新增数据暂存在浏览器状态，刷新页面后重置；闽南语词库也需要按具体地区和家庭习惯核对。
